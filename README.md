# Sistema de Ordem de Serviço

## Tecnologias Utilizadas

* PHP
* HTML
* CSS
* JavaScript

## Sobre o Projeto

Este projeto foi iniciado durante meu estágio, sendo desenvolvido em conjunto com outros colegas que também atuam e estudam programação.

A proposta era criar um sistema interno de Ordem de Serviço, estruturado para futura integração com banco de dados corporativo. A integração não foi concluída por questões internas da empresa, permanecendo a aplicação em versão simplificada para uso interno.

A versão disponibilizada neste repositório representa a estrutura base do sistema, com foco em autenticação, organização de páginas e separação de responsabilidades.

## Experiência de Aprendizado

Este projeto marcou meu primeiro contato com JavaScript.
A implementação foi realizada com base em pesquisas, documentação e estudos práticos, aplicando conceitos básicos para auxiliar na interface e validação de campos.

O desenvolvimento também proporcionou experiência em:

* Trabalho em equipe
* Estruturação inicial de sistema web
* Organização de arquivos e separação de camadas
* Implementação de autenticação com sessão

## Funcionalidades

* Sistema de login com autenticação de administrador
* Alteração de senha
* Controle de acesso utilizando sessão
* Interface estilizada com CSS separado por página
* Uso básico de JavaScript para suporte à interface

## Objetivo

Este repositório tem como finalidade apresentar a experiência prática adquirida durante o estágio, demonstrando evolução técnica, capacidade de aprendizado e participação em desenvolvimento colaborativo.
